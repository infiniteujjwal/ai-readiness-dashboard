Embedding Methods

The simplest way to embed - just copy and paste this iframe code:

<!-- Simple iframe embed --> <iframe src="http://localhost:3000/embed" width="100%" height="900px" frameborder="0" style="border: none; border-radius: 8px;" ></iframe>

With customization options:

<!-- Customized embed --> <iframe src="http://localhost:3000/embed?hideHeader=true&compact=true" width="100%" height="700px" ></iframe>

Use our embed script for more control:

<!-- Container for the dashboard --> <div id="ai-readiness-dashboard" data-height="900px" data-hide-header="false" data-compact="false"></div> <!-- Load the embed script --> <script src="http://localhost:3000/embed.js"></script>

Full JavaScript API for programmatic control:

// Configure before loading window.AIReadinessConfig = { containerId: 'my-dashboard', height: '800px', hideHeader: false, onLoad: function() { console.log('Dashboard loaded!'); }, onDataChange: function(data) { console.log('Data updated:', data); } };
